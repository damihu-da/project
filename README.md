# project

## 贪吃蛇

静态前端小游戏，无需构建。

- 入口：[snake/index.html](snake/index.html)（或从根目录 [index.html](index.html) 跳转）
- 操作：方向键 / WASD 移动，空格暂停；小屏幕可使用方向按钮
- 最高分保存在浏览器 `localStorage`

本地预览：用浏览器打开 `snake/index.html`，或在项目根目录执行：

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080/snake/`。

## GitHub Pages

仓库启用 Pages 后，游戏地址一般为：`https://<用户名>.github.io/<仓库名>/snake/`
